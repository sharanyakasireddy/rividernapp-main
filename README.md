Rivide is a real-time carpooling app built using React Native, Firebase, and the Google Maps API. It allows users to securely sign up, offer, or find rides and view available routes in real time.

The app helps reduce travel costs and traffic congestion while promoting eco-friendly commuting. Built with Firebase Authentication for secure login and Firestore for real-time data sync, Rivide provides a smooth and responsive user experience for everyday travel.
